Question:

If you were writing a "Profanity Check" program, what steps would you take to get to the output?

Answer:

i would first define a function. And in that function:

  1. create a list with cursed word that I would like to avoid.
  2. parse the words from the given text and put them into a list.
  3. iterate over each word.
  4. compare the words with the items in cursed word list
  5. if there is any match
  6. raise an alert
  7. if there isn't
  8. say that everything is OK.
